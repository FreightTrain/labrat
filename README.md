# What is Lab Rat

This is an example CloudFoundry app that used to inspect
values of environment variables in a Warden container.


## How to Run It

    bundle install
    bundle exec puma --config puma.rb

## How to Deploy

    # assuming you've done cf target and cf login
    # earlier
    cf push
